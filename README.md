## Mô tả project:

Đây là project xây dựng một ứng dụng web là chiếc máy tính có thể tính toán các phép tính cơ bản áp dụng kiến thức: `HTML`, `CSS`, `Git` và `Github`.

## Các chức năng:

- Thực hiện các phép tính cơ bản:
  - Phép cộng
  - Phép trừ
  - Phép nhân
  - Phép chia
- Thay đổi giao diện:
  - Dark mode
  - Light mode
